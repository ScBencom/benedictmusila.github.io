ASSETS FOLDER
==============

This folder is for storing your CV and other assets (images, documents, etc.)

REQUIRED FILE:
--------------
Please add your CV as a PDF file with the exact name:

    Benedict_Musila_CV.pdf

This will enable the "Download CV" button on the website to work properly.

OPTIONAL FILES:
---------------
You can also add:
- Professional headshot photo (e.g., profile.jpg)
- Company logos
- Certificate images
- Any other supporting documents

TIPS:
-----
1. Keep file sizes optimized for web (compress images)
2. Use descriptive filenames (no spaces, use hyphens or underscores)
3. Supported formats: PDF, JPG, PNG, WebP
4. Recommended image size: max 1MB per image

Once you add Benedict_Musila_CV.pdf here, the download button will work automatically!
